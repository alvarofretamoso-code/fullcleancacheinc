# FULL CLEAN CACHE IN C:

Escanea Windows en busca de carpetas de caché, calcula cuánto ocupan y te deja borrarlas o enviarlas a la Papelera.

## Uso

```bash
python limpiar_cache.py
```

Elige carpeta (C:\, explorador o carpeta actual), revisa el informe (`.csv` / `.txt`) y decide: **eliminar**, **enviar a la Papelera** o **salir**.
